Set up Wallet Whisperer on your machine.

Three short stages. You'll install the onchainos backbone, then either the standalone CLI, the agent skill, or both.

1
Stage 1

Install prerequisites

You need Node.js 18+ and the onchainos CLI. Both are quick.

Node.js (check first)
$ node --versioncopy

If you don't have Node 18+, install via nodejs.org or use nvm install 20.

Onchainos CLI (the data backbone)
$ curl -sSL https://raw.githubusercontent.com/okx/onchainos-skills/main/install.sh | shcopy

Installs the binary to ~/.local/bin/onchainos. Wallet Whisperer wraps this CLI; nothing else talks to OKX.

Make sure your PATH picks it up
$ export PATH="$HOME/.local/bin:$PATH"copy

Add that line to your ~/.zshrc or ~/.bashrc so it sticks.

Verify
$ onchainos --versioncopy
2
Stage 2

Log in to onchainos

Free for the first 1,000,000 API calls / month. No payment, no wallet funding, no signature needed.

Request a one-time code
$ onchainos wallet login your@email.comcopy

Check your inbox for the verification code.

Enter the code
$ onchainos wallet verify 123456copy
Confirm
$ onchainos wallet statuscopy

Should show "loggedIn": true.

3
Stage 3

Pick a surface

Clone the repo once, then install whichever interface you want. All three share the same persona engine.

Clone the repo
$ git clone https://github.com/Temitope15/wallet-whisperer.git && cd wallet-whisperercopy

Everything below assumes you're inside this directory.

Surface A

CLI (terminal)

~30 seconds

For one-shot lookups, scripts, and CI.

Install globally
$ cd cli && npm install -g .copy
Profile a wallet
$ wallet-whisperer whisper 21czpZj3BxT75dVbzmUJtE5QznJrLrYHHaF5pT4CpWM1copy
Other commands: wallet-whisperer replay <addr>, wallet-whisperer mirror <addr>, --help.
Surface B

Agent skill (Claude Code / Cursor / …)

mirror enabled

Adds the interactive mirror loop with one-tap per-trade confirmation inside your agent host.

One command for every host
$wallet-whisperer init claude

Surface C

Web app (local)

visual

Zero-dependency Node server, opens at localhost:4444. Same engine as the CLI, plus a live OKX skills sidebar.

Start the server
$ cd cli && npm run webcopy
Open in your browser
→http://localhost:4444copy
